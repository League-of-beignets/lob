This folder contains your original .htaccess file before the aeSecure protection.
If you wish to uninstall aeSecure, rename the old_xxxxxx.htaccess file that you'll
find here, in the folder, and save the file in your website root folder.  His name
should be .htaccess (no name, just the .htaccess extension).

Ce dossier reprend votre fichier .htaccess d'origine; avant la protection apportée
par aeSecure.  Si vous souhaitez désinstaller aeSecure, renommer le fichier
old_xxxxxxx.htaccess que vous trouverez ici et sauver le fichier dans le dossier
racine de votre site web.  Le nom du fichier devra être .htaccess (càd pas de nom,
juste l'extension .htaccess)
